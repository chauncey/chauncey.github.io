Stuff I like and use

Python

Python enables quick prototyping with a ton of useful modules in the standard library. It's a powerful language with an easy to read syntax.

Cyclone

Built on Python's Twisted library to work like Tornado web server. I like that it's built on Twisted so that deep down you have a huge arsenal of goodies to use.

Flask

Python microframework with all plenty of goodies without imposing a set structure. I find this a well thought out framework that gives you the structure to build quickly without getting in your way.

Node.js

Fast growing ecosystem with a language that's suited for frontend or backend development. Node.js makes it easy to write scripts that run async processes in a natural way.
